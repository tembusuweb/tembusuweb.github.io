About

Tembusu Web Design Workshop is planned by two Tembusu Computing students, Tan Kang Soon and Andhieka Putra. The purpose is to share their passion for programming and web design to fellow Tembusians.

The workshop will teach participants to design a simple website using HTML, CSS and Bootstrap templates, as well as how to host websites using GitHub Pages. Register now if you haven't done so!

Sublime Text 3

Sublime Text is an excellent user-friendly text editor for Mac, Windows and Linux. Download the latest Sublime Text 3 version here

SourceTree

SourceTree is an easy-to-use Git client. It is really handy in this workshop to push our website for hosting in GitHub. Download the latest version here.

Google Chrome

One of the most popular web browser, it needs no introduction. Google Chrome adheres very closely to the latest HTML and CSS standard. The workshop requires you to use Google Chrome to ensure that your webpage appears correctly as you write it. Download the latest here.

Bootstrap

Bootstrap is a pre-cooked CSS and JavaScript files to accelerate the process of desigining new websites. It provides a sensible default look to your website. Download Bootstrap here.
